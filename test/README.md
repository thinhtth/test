# Project Documentation

## TestApp

This project is a simple Java application that demonstrates the basic structure of a Java program.

### Purpose

The purpose of this application is to print a greeting message to the console. It serves as an introductory example for those learning Java.

### How to Run the Application

1. Ensure you have Java Development Kit (JDK) installed on your machine.
2. Navigate to the project directory in your terminal.
3. Compile the Java file using the following command:
   ```
   javac src/TestApp.java
   ```
4. Run the application with the command:
   ```
   java -cp src TestApp
   ```

You should see the output:
```
Hello, Java App!
```
